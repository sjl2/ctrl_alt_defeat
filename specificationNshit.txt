~~~~~ Playmaker ~~~~~

Save Handler --

Front End: Send post parameters in the form (for example):
	name : "Pick and Roll"
	numFrames : "50"
	paths : <stringify Location[][]>

Back End: Parses play info, sends back list of play names:
	plays : List<String>

	(So that you can display the full list of plays to the user after saving)


Load Handler --

Front End: Send the name of the play you want:
	name : "Pick and Roll"

Back End: Parses name, retrieves from the DB, responds with:
	"play" : <Stringified Location[][]>



Delete Handler --

Front End: Send the name of the play you want to delete to /playmaker/delete, in the same way as the others

Back End: Parses name, deletes from database, responds with:
	"plays" : List<String>, for updating the list of plays.



GetPlayNamesHandler -- 

Front End: Just a get request, no paramters

Back End: Response with list of play names:
	"plays" : List of Strings
